# Changelog

All notable changes to Miniverto will be documented in this file.

## 0.1.0-alpha

Initial open-source alpha preparation.

- Added the React and Tauri desktop app foundation.
- Added local project planning, milestones, tasks, notes, materials, and assessments.
- Added local SQLite stores and Rust unit tests for core persistence behavior.
- Added provider configuration and OS keyring storage commands.
- Added deterministic plan generation fallback for offline or unconfigured model states.
- Added public README, license, contribution guide, security policy, roadmap, architecture notes, and CI workflow.
