## Short Description

Local-first AI learning planner for turning goals, materials, and self-assessments into adaptive study plans.

## Longer About Text

Miniverto is an open-source desktop app for self-directed learners who need more than a static syllabus. It helps turn a vague learning goal into milestones, tasks, lesson flows, notes, and assessment loops while keeping user data local by default.

Author: kevin tao.
