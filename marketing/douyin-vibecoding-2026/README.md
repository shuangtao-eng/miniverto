# Miniverto 抖音 VibeCoding 投稿包

这个目录是给「VibeCoding大赏」准备的中文投稿素材包，目标不是刚好满足最低要求，而是做成一组更适合抖音图文和视频传播的作品展示素材。

## 活动要求

根据你提供的活动图，投稿需要注意：

- 标题建议以 `VibeCoding大赏` 开头，更容易被官方捕捉。
- 必带话题：`#vibecoding大赏`、`#ai新星计划`。
- 必须 @：`@抖音科技`。
- 投稿方向：展示作品与亮点，分享落地过程与思考。
- 图文形式：文字不少于 100 字，图片不少于 3 张。
- 视频形式：视频时长不少于 1 分钟。
- 征集时间：即日起至 6 月 30 日。

## 推荐发布内容

- 图文主文案：`douyin-post-copy.md`
- 项目核心介绍：`project-core.md`
- 抖音视频脚本：`video-script-douyin.md`
- 发布前检查清单：`checklist.md`
- 图片素材：`images/`
- 图片生成脚本：`create_share_images.py`
- 视频生成脚本：`create_douyin_video.py`
- 最终视频：`video/miniverto-vibecoding-douyin-vertical.mp4`

## 推荐发布策略

先发图文，因为图文更适合完整讲清楚产品定位和开源地址。视频可以随后发布，用同一套标题和话题。当前视频已经包含中文演示界面、功能放大讲解、中文字幕和原创 BGM。

图文发布时建议把 GitHub 链接放在正文和第一条评论里。如果链接不可点击，就在评论里写：

```text
GitHub 搜索：shuangtao-eng/miniverto
```
